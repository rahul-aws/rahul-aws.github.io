# Rahul Kumar — DevOps Engineer Portfolio

This is the source code for my personal DevOps portfolio website, showcasing my skills, experience, and projects.

## 🚀 What’s Inside

- A responsive developer portfolio website
- Icons and logos for AWS, GCP, Helm, Kubernetes, Git, Jenkins, Docker, and more
- Hosted on GitHub Pages

## 📦 Technologies Used

- HTML5, CSS3
- Jekyll (for GitHub Pages support)
- Devicon for tool logos
- Font Awesome
- Minimal JS (optional)
